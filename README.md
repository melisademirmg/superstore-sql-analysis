\# Superstore Sales Analysis Using SQL



\## Project Overview



This project analyzes retail sales data using SQL in Google BigQuery.



The goal of the project is to explore sales performance, customer behavior, product profitability, and business insights using real-world retail data.



\---



\## Tools Used



\- Google BigQuery

\- SQL

\- CSV Dataset (Superstore)

\## Data Quality Checks



The following validations were performed:



\- Row count verification

\- Unique customer count

\- Unique order count

\- NULL value checks

\- Category consistency checks

\- Row ID uniqueness validation



\---



\## Key Findings



\- Technology generated the highest sales revenue.

\- Technology produced the highest total profit.

\- Furniture had the lowest profit margin.

\- The most profitable product was the Canon imageCLASS 2200 Advanced Copier.

\- The highest spending customer was Sean Miller.

\- The most profitable customer was Tamara Chand.
